# React Taproom

React application that manages inventory of a taproom, with fields to view, add, and edit kegs.

### Installing

Clone this repo

```
git clone ...
```

navigate to project

```
cd ...
```

install npm

```
npm install
```

launch project in your in browser

```
ng serve --open
```
## Languages Used

* JavaScript / JSX
* HTML/CSS
